🔋 Battery First

Cheap hour + low solar. Stock up from grid at low price.

⚡ Grid First

Expensive hour + low solar. Discharge battery, avoid import.

☀️ Load First

Solar hour or neutral. Self-consume solar, don't force charge.

Row Level Security enabled on all tables. Public SELECT, INSERT, UPDATE permitted — personal single-user dashboard, no multi-user auth required. Anon key stored as Vercel environment variable, never exposed in source code.

Security
🔑
Credentials
GROWATT_USER, GROWATT_PASS, SUPABASE_URL, SUPABASE_ANON_KEY stored as Vercel env vars. Never in source code or client.
🛡️
Row Level Security
Enabled on all Supabase tables. Anon key allows read + write but nothing outside defined RLS policies.
🔒
TOU write gate
Growatt inverter writes require a password entered in the UI. Validated server-side before any TOU segment is pushed to the inverter.
📅
Data integrity guard
save_summary.py has a server-side CEST date check — refuses writes for today or future dates regardless of client behaviour or stale cache.
🌐
CORS
All API handlers return Access-Control-Allow-Origin: *. Acceptable for a personal tool with no sensitive write endpoints exposed without auth.
⏱️
Growatt rate limiting
live.py reads from Supabase instead of Growatt, eliminating ~4 000 direct API calls per day and reducing account lockout risk.
Key Design Decisions
DecisionRationale
Single index.html, no build stepZero toolchain — instant deploy, trivial to inspect, works in any browser without npm.
Vercel Hobby + cron-job.org hybridVercel Hobby only supports daily crons natively. 5-min live data requires an external scheduler.
Supabase anon key, no user authPersonal single-user dashboard. Full auth would add complexity with zero benefit.
eSett instead of ENTSO-EeSett requires no token and returns clean JSON. ENTSO-E requires registration and returns XML.
Nuclear nominal = 6 804 MW hardcodedForsmark 1+2+3 + Ringhals 3+4 + Oskarshamn 3. Changes only if reactors open or permanently close.
12-function Vercel limit awarenessDrove deletion of backfill.py to add grid.py. Helper modules use _ prefix to avoid counting.
−15% solar forecast haircutSafety margin on TOU suggestion to avoid over-relying on solar that may not materialise (clouds, dust, shading).
